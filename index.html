Qt BItcoin Trader

Qt Bitcoin Trader Mt.Gox Client

Download this project as a .zip file Download this project as a tar.gz file

Description

This software helps you to open and cancel Mt.Gox orders vary fast. Real time data monitoring. Developed on pure Qt, uses OpenSSL, AES 256 key and secret protection.

I want to develop Trader App that can be configured for any rule and strategy. If you want to help me, than donate to 1d6iMwjjNo8ZGYeJBZKXgcgVk9o7fXcjc
